## 🌱(MennCare) Mental Wellness Website 

This project is a mental health and therapy-focused website built using:

- ⚛️ **React** for interactive UI
- 🎨 **Tailwind CSS** for fast and responsive styling
- 📩 **Getform.io** to handle contact form submissions without backend code
- 🔔 **React Toastify** for user-friendly success/error notifications

### 🧠 Features

- Informative content about therapy and mental wellness
- Appointment booking form with real-time validation
- Mobile-friendly and fully responsive design
- Smooth user experience with feedback on submission
